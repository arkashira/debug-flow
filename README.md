# Debug Flow

A small, REAL, runnable Python project for a visual debugger engine.

## Features

* Set, move, and delete breakpoints on any line
* Pause execution at breakpoints and highlight the active node in the flow-chart
* Variable watch panel shows current locals/globals for the paused frame
* Stepping (into/over/out) updates both code pane and flow-chart synchronously

## Usage

1. Install the project using `poetry install`
2. Run the tests using `pytest`
3. Use the `debug_flow` module in your own projects
