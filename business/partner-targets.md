# Partner Targets

## Integration Roadmap

### 1. GitHub
- **Rationale**: GitHub is the most popular platform for hosting and collaborating on code. Integrating with GitHub will allow users to easily share and collaborate on debug sessions directly from their repositories.
- **Free-tier limits**: GitHub's free tier allows for unlimited public repositories and 500MB of storage per repository.
- **Integration effort**: M (Medium)
- **Value-add**: Enables seamless collaboration and version control for debugging sessions.
- **Affiliate/Revenue-share**: Potential for affiliate revenue through GitHub's developer program.

### 2. Jupyter Notebooks
- **Rationale**: Jupyter Notebooks are widely used in scientific programming for data analysis and visualization. Integrating with Jupyter will allow users to debug their notebooks directly within the platform.
- **Free-tier limits**: Jupyter Notebooks are open-source and free to use.
- **Integration effort**: S (Small)
- **Value-add**: Simplifies the debugging process for data scientists and researchers.
- **Affiliate/Revenue-share**: No direct affiliate program, but potential for partnerships with Jupyter ecosystem tools.

### 3. Slack
- **Rationale**: Slack is a popular communication tool for teams. Integrating with Slack will allow users to share debug sessions and collaborate in real-time within their Slack channels.
- **Free-tier limits**: Slack's free tier allows for 10,000 messages and 10 integrations.
- **Integration effort**: M (Medium)
- **Value-add**: Enhances team collaboration and communication during debugging sessions.
- **Affiliate/Revenue-share**: Potential for affiliate revenue through Slack's developer program.

### 4. Google Colab
- **Rationale**: Google Colab is a popular cloud-based platform for running Jupyter Notebooks. Integrating with Google Colab will allow users to debug their notebooks directly within the platform.
- **Free-tier limits**: Google Colab offers free access to GPUs and TPUs with some usage limits.
- **Integration effort**: M (Medium)
- **Value-add**: Simplifies the debugging process for users who prefer cloud-based environments.
- **Affiliate/Revenue-share**: No direct affiliate program, but potential for partnerships with Google Cloud tools.

### 5. VS Code
- **Rationale**: VS Code is a popular code editor with a large user base. Integrating with VS Code will allow users to debug their code directly within the editor.
- **Free-tier limits**: VS Code is open-source and free to use.
- **Integration effort**: L (Large)
- **Value-add**: Enhances the debugging experience for users who prefer using VS Code.
- **Affiliate/Revenue-share**: Potential for affiliate revenue through Microsoft's developer program.

### 6. Trello
- **Rationale**: Trello is a popular project management tool. Integrating with Trello will allow users to track and manage debugging tasks within their Trello boards.
- **Free-tier limits**: Trello's free tier allows for unlimited boards and lists.
- **Integration effort**: S (Small)
- **Value-add**: Enhances task management and collaboration for debugging projects.
- **Affiliate/Revenue-share**: Potential for affiliate revenue through Trello's developer program.

### 7. Zapier
- **Rationale**: Zapier is a popular automation tool that connects different SaaS applications. Integrating with Zapier will allow users to automate workflows involving debug sessions.
- **Free-tier limits**: Zapier's free tier allows for 100 tasks per month.
- **Integration effort**: M (Medium)
- **Value-add**: Enhances automation and integration capabilities for debugging workflows.
- **Affiliate/Revenue-share**: Potential for affiliate revenue through Zapier's partner program.

### 8. AWS Lambda
- **Rationale**: AWS Lambda is a popular serverless computing service. Integrating with AWS Lambda will allow users to debug their serverless functions directly within the platform.
- **Free-tier limits**: AWS Lambda offers 1 million free requests per month.
- **Integration effort**: L (Large)
- **Value-add**: Simplifies the debugging process for serverless applications.
- **Affiliate/Revenue-share**: Potential for affiliate revenue through AWS's partner program.
